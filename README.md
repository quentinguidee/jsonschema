# `jsonschema`

An implementation of [json-schema](https://json-schema.org/specification) in [Go](https://go.dev/).

## Usage

```go
import "github.com/quentinguidee/jsonschema"

s := jsonschema.Schema{}
```

The schema is compatible with
- `encoding/json`
- `gopkg.in/yaml.v3`
